Nomad

// headline
Esculpe, pinta y crea en 3D

// tag
esculpir, 3D, pintar

---------------------------------------------------------------------

- Herramientas de escultura
Los pinceles de arcilla, aplanar, alisar, enmascarar, entre otros, te permitirán dar forma a tu creación.
¿Necesitas líneas duras? También encontrarás algunas herramientas de corte de lazo/línea/curva.

- Personalización de trazos
Se pueden personalizar la caída, los alfas, los mosaicos, la presión del lápiz y otros parámetros de trazo.
También puedes guardar y cargar tus ajustes preestablecidos de herramientas.

- Herramientas de pintura
Pintura de vértices con color, rugosidad y metalicidad.
También puedes administrar fácilmente todos tus ajustes preestablecidos de materiales.

- Capas
Graba tus operaciones de escultura y pintura en capas separadas para facilitar la iteración durante el proceso de creación.
Se graban tanto los cambios de escultura como de pintura.

- Escultura multirresolución
Cambia de una resolución a otra con tu malla para un flujo de trabajo flexible.

- Remezcla de vóxeles
Refuerza la malla rápidamente para obtener un nivel de detalle uniforme.
Se puede usar para dibujar rápidamente una forma aproximada al comienzo del proceso de creación.

- Topología dinámica
Refina localmente la malla debajo del pincel para obtener un nivel de detalle automático.
Incluso puedes conservar tus capas, ya que se actualizarán automáticamente.

- Forma primitiva
El cilindro, el toro, el vóxel triplanar y otras formas primitivas se pueden usar para comenzar rápidamente una nueva forma desde cero.

- Renderizado PBR
Hermoso renderizado PBR por defecto, con iluminación y sombras.
Siempre puedes cambiar a matcap para obtener un sombreado más estándar para esculpir.

- Posprocesado
Reflexión del espacio de la pantalla, profundidad de campo, oclusión ambiental, mapeo de tonos, etc

- Archivos
Guarda y abre archivos glTF, OBJ y STL.
Las capas y la pintura PBR solo se conservan a través de archivos GLTF.

- Interfaz
Interfaz fácil de usar, diseñada para la experiencia móvil.
¡También puedes personalizar la interfaz!
